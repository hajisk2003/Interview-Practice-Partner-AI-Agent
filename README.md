# Interview Practice Partner

A simple web-based agent to practice interviews via chat and voice.

---

## How It Works

User (Browser)
│

Frontend (HTML + JS + Web Speech API)
│ Sends user input (text/voice)

Backend (FastAPI)
│ Sends prompts to LLM (OpenAI API)

LLM (OpenAI) generates questions & feedback
│

Backend returns response
│

Frontend displays answer / plays voice

---

## Features

- Chat & voice-based interview practice
- Role selection: Software Engineer, Data Analyst, Sales
- Automatic follow-up questions & scoring
- Export session transcript as `.txt`
